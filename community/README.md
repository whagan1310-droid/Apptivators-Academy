# 👥 Community Guidelines

Welcome to Apptivators Academy! This community is dedicated to helping developers learn, grow, and build amazing applications.

## Core Values
- **Respect** - Treat everyone with respect
- **Learning** - Share knowledge freely
- **Building** - Focus on practical projects
- **Protection** - Keep the community safe

## Rules
1. No spam or self-promotion
2. No NSFW or offensive content
3. No piracy or illegal content
4. Help others learn and grow
5. Follow Discord ToS

## Getting Started
1. Complete the onboarding carousel
2. Choose your path (The Forge progression)
3. Start building!

*"One App At A Time. Build it. Share it. Protect it."* ⚔️🛡️🤖
